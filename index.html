Ping-Pong Game

Un joc classic de ping-pong realizat folosing WebGL (Web Graphics Library), libraria nativa JavaScript pentru redarea graficelor interactive 3D și 2D cu performanțe ridicate în orice browser web compatibil, fără a utiliza plugin-uri.
WebGL realizează acest lucru introducând un API care se conformează strâns cu OpenGL ES 2.0 și care poate fi utilizat în elementele HTML canvas.
